## Phonetic Service Center – CLI Simulation

### Requirements

* Node.js ≥ 18
* Yarn

### Install

```bash
yarn install
```

### Build

Compile TypeScript to JavaScript:

```bash
yarn build
```

### Run

Run the CLI simulation:

```bash
yarn start
```

### Test

Run unit tests:

```bash
yarn test
```

### Notes

* The app simulates two technicians working **concurrently**.
* Repair times are scaled for simulation purposes.
* Processing flow is printed in real time, with a **final service log** shown at the end.
